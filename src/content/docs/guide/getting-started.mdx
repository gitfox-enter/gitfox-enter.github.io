# 海阔视界新手指南

海阔视界是一款强大的安卓端多功能工具应用，集浏览器、小说阅读、视频播放、规则扩展于一体。

> 本文内容来自海阔视界官方博客 [haikuoshijie.cn](https://haikuoshijie.cn/)，转载供学习参考。

## 什么是海阔视界？

海阔视界是一个基于 WebView 的安卓应用，其核心特色是**强大的扩展能力**——用户可以通过编写 JavaScript 规则来扩展功能，包括但不限于：

- 网页规则：自定义站点适配
- 小说规则：自定义小说源
- 视频规则：自定义视频播放器
- 小程序：内置各类实用工具

## 基础使用

### 安装与更新

1. 从官方渠道下载 APK 安装包
2. 首次启动授予必要权限（存储、网络等）
3. 定期检查应用内更新或从公众号获取最新版本

### 界面介绍

- **首页**：搜索入口、常用站点入口
- **发现页**：推荐规则、热门内容
- **小程序**：内置工具集合
- **设置**：应用配置、规则管理

## 规则系统

### 什么是规则？

规则是由社区用户编写的 JavaScript 代码，用于适配各类网站。通过规则，你可以：

- 为任意网站添加自定义 UI
- 提取网页内容（文本、图片、视频）
- 实现自动翻页、无广告阅读

### 如何导入规则？

1. 打开海阔视界
2. 进入"规则管理"
3. 点击"导入"或扫描规则二维码
4. 选择本地 .js 文件或粘贴在线链接

### 常用规则源

- 公众号"新方圆小棉袄"每月推荐
- 品技论坛海阔视界版块
- 官方 Telegram 频道

## 小说阅读

海阔视界内置强大的小说阅读器：

- 支持自定义书源
- 支持语音朗读（TTS）
- 支持字体、背景、行距等个性化设置
- 支持离线缓存

## 视频播放

- 支持 M3U8、MP4 等主流格式
- 支持倍速播放
- 支持悬浮窗播放
- 支持投屏

## 进阶技巧

### 备份与恢复

- 定期备份配置和规则
- 通过云盘或本地文件恢复

### 小程序扩展

海阔视界的小程序系统支持：

- 本地 HTML 小程序
- 在线加载远程小程序
- 各类实用工具集成

## 常见问题

**Q: 规则失效怎么办？**
A: 优先检查是否有更新版本的规则，或联系规则作者获取支持。

**Q: 为什么有些网站打不开？**
A: 部分网站可能需要特殊网络环境。可以尝试切换 User-Agent 或检查规则配置。

**Q: 如何反馈 Bug？**
A: 通过公众号"新方圆小棉袄"留言，或加入官方用户群反馈。
